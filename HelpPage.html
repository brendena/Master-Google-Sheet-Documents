First make a new folder and put all the similar files you want to be merged together

Then select the file type in the white nav bar and a document ui picker will come up. Then select the main document you want all the other files to be merged into"

Once its done your'll have one file and you will see a new yellow row on the bottom of the navigation.

If you hit the refresh icon it will go through each of the files that are in the yellow column and merge all the newly added files into the main file.
